# art ✦

AI image generation covers character posts, abstract art,
and complex scene compositions. in this system three tools
handle different creative needs — Grok Imagine for character
consistency and abstract art, GPT Image 2.0 for complex scenes
and thumbnails with text, and Higgsfield for video-forward
visual content.

---

## tools in this workflow

| tool | role |
|------|------|
| Grok Imagine | character posts, abstract surreal art |
| GPT Image 2.0 | complex scenes, thumbnails with text overlay |
| Higgsfield | AI video generation, branded product content |
| Claude | writes image prompts, maintains character spec |
| Obsidian | stores character spec and visual brand guidelines |

---

## two content types — never mix them

**dark cinematic character:**
a consistent visual identity across posts. same character,
same aesthetic, different scene or mood each time.

character spec:
- caramel skin (not warm brown — comes out too dark)
- long dark hair, straight or wavy — never short
- small gold nose stud
- form-fitting clothing only — no oversized
- dark mode always — deep blacks, purples, midnight blues, neons
- art style: cinematic digital art

caption style: short and conversational, or a question

**abstract surreal art:**
no character — pure concept. skeletons, cosmic scenes,
glitch aesthetics, liquid paint, transformation imagery.
things glowing in the dark. death as beautiful.

caption style: one philosophical line. no explanation.

## Grok Imagine workflow

1. decide which type: character or abstract
2. Claude writes the prompt using the character spec stored
   in the vault (for character posts) or the concept direction
   (for abstract)
3. generate in Grok Imagine
4. pick the output that matches the spec
5. pair with caption — Claude drafts both prompt and caption
   together

for abstract art: both Grok Imagine and GPT Image 2.0 produce
strong results. test both on the same concept when time allows.

---

## GPT Image 2.0 workflow

best for:
- thumbnails that need text rendered correctly in the image
- multi-element scenes (character + screens + environment)
- photorealistic UI mockups
- complex compositions requiring precise control

workflow is the same as Grok Imagine — Claude writes the prompt,
you generate, you pick.

## Higgsfield workflow

Higgsfield handles video-forward creative content. see

produces strong branded product images — useful when you
have a product to feature rather than a character or concept.
