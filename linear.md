# Linear 📋

Linear is a project and issue tracking tool. in this system
it functions as the execution layer — every task, workflow,
and idea that needs to get done lives in Linear as an issue.
it has an MCP connector, which means Claude can create, update,
and close issues mid-session without leaving the conversation.

---

## how you can use it

- track tasks and projects across every area of your work
- organize work into teams, projects, and cycles
- set priorities and track status (backlog, in progress, done)
- connect to Claude via MCP so task management happens inside
  your AI sessions — no context switching

---

## setup

1. create an account at [linear.app](https://linear.app)
2. create a team for your work ([your team name])
3. enable the Linear MCP in Claude Desktop under
   Settings → Integrations

---

## how i use it

every session ends with Linear updated. if a task gets done,
Claude marks it done. if something new surfaces mid-session,
Claude creates an issue on the spot with the right priority,
project, and labels.

the vault and Linear are bidirectionally linked. when i capture
a monetizable idea in the vault (as an Idea Card), Claude creates
a corresponding Linear issue at the same time. the vault file
references the Linear ID. the Linear issue links back to the
vault file. nothing falls through the cracks between the two
systems.

current projects in Linear:
- Content (daily posts, campaigns, threads)
- AI Workflows (builds, documentation, experiments)
- Monetizable Ideas (captured ideas with revenue potential)
- Music (releases, distribution, registration)
- DNA of a Rose (novel — chapters, research, character work)

Claude pulls open issues at the start of every session,
checks what's in progress, and closes completed items
automatically as work gets done.
