# workflow doc template

use this template to document any workflow you build. one file
per workflow. saved in your vault. the act of documenting it
is what makes it a product — not just a process you run once
and forget.

every workflow documented here is simultaneously:
- a tool (you can run it again)
- a content piece (you can post about it)
- a product (you can sell it)

## example

```markdown
# Lyric-to-Music-Video (Grok Imagine Agent Mode)

**Status:** Documented
**Tools used:** Grok Imagine (Agent Mode), CapCut
**Time to run:** 30-45 minutes
**Task ID:** AUN-188

---

## what it does

Produces a full music video from song lyrics in a single
Grok Imagine Agent Mode conversation. output: a stitched
video file synced to the track, ready to post.

---

## who it's for

Musicians and creators who want music video content without
a production budget or video editing skills.

---

## step-by-step

1. paste your lyrics into Grok Imagine Agent Mode
2. request a scene-by-scene storyboard (one scene per section)
3. Grok generates animated clips for each scene
4. request B-roll clips for instrumental gaps
5. download all clips
6. stitch in CapCut — arrange in order, sync to audio
7. export and post

---
```
