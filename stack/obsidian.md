# Obsidian 📓

Obsidian is a note-taking app that stores everything as plain
markdown files on your device. in this system it functions as the
memory layer — a structured vault that every AI tool reads from,
so nothing has to be re-explained between sessions.

---

## how you can use it

- store your voice, goals, and context so AI tools always have
  the full picture
- build session protocols that Claude reads before every task
- track projects, ideas, workflows, and decisions in one place
- connect to Claude via MCP so it can read and write files
  mid-session
- sync across devices via iCloud or GitHub so your vault is
  always current
- build templates for recurring tasks — content batches, idea
  cards, session logs

---

## setup

1. download Obsidian at [obsidian.md](https://obsidian.md) — free,
   available on mac, windows, ios, android
2. create a new vault (this is just a folder on your device)
3. set up your folder structure
   (see [vault-structure/folder-map.md](../vault-structure/folder-map.md) for a starting point)
4. write your core files — a content manual, a session protocol,
   a links file. these are what Claude reads first.
5. connect to Claude via MCP
   (see [mcp-setup/setup_guide.md](../mcp-setup/setup_guide.md))
6. sync to iCloud or GitHub so your vault stays current across
   devices

---

## how i use it

my vault has over 200 files across every area of my work —
content strategy, music releases, consulting, novel writing,
AI workflows, monetization tracking. it's organized into
emoji-prefixed folders so Claude can navigate it precisely.

the files Claude reads every session:
- `_index.md` — the master map of the whole vault
- `🕷️ Black Widow HQ.md` — active projects and routing
- `📝 Session Log.md` — what changed across recent sessions
- domain files for whatever we're working on that day

when Claude reads these before doing anything, it knows:
- what i'm currently building
- what's already been written (no repeats)
- what my voice sounds like
- what tools are available
- what's open in Linear

the vault also stores every prompt pattern, workflow doc, and
session output — so the system gets smarter the more i use it.
the folder structure template is in [vault-structure/folder-map.md](../vault-structure/folder-map.md)
