# DistroKid 🎧

DistroKid distributes music to every major streaming platform —
Spotify, Apple Music, YouTube Music, Amazon Music, Tidal, and
more. you upload your track once and it goes everywhere.
it also hosts hyperfollow pages — pre-save and streaming link
pages for each release.

## how you can use it

- distribute music to all major streaming platforms in one upload
- keep 100% of your royalties
- create hyperfollow pages for pre-saves and release links
- manage your full catalog from one dashboard
- collect streaming royalties across platforms

---

## setup

1. create an account at [distrokid.com](https://distrokid.com/vip/seven/11119560)
2. choose a plan based on how many releases you plan per year
3. upload your track — WAV or FLAC, with artwork
4. fill out metadata (title, genre, release date, ISRC)
5. submit and it goes live within 24-72 hours on most platforms

## how i use it

every track goes through the same pipeline before hitting
DistroKid:

```
Suno generates → Soundboost AI masters →
DistroKid distributes → hyperfollow page built →
streaming links added to vault
```

the hyperfollow pages used to be manual. now Claude handles
them — it reads the release details from the vault and builds
each page autonomously via Claude Cowork.

i also use DistroKid's ISRC codes and catalog data as inputs
for ASCAP registration. Claude Cowork registered my full
150-song catalog on ASCAP in a single session using the
DistroKid export.

every streaming link gets saved to the vault after distribution
so the full catalog stays current and accessible for any
future task that needs it.
